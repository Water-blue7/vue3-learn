## 创建vue项目的方式有好几种，推荐使用Vite创建
>Vite 提供了快速的开发环境、简单的配置和强大的功能，是创建和开发 Vue 3 项目的绝佳选择。

1. 安装 Vite：
通过 npm 安装 Vite：

```bash
npm create vite@latest my-vue3-project
```
1. 选择模板：
在交互式提示中，选择 vue 或 vue-ts（如果你想使用 TypeScript）。

1. 安装依赖：
进入项目目录并安装依赖：

```bash
cd my-vue3-project

# 安装依赖
npm install
# 或者
npm i
```


4. 启动开发服务器：

```bash
npm run dev
```
